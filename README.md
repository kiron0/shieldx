<h1 align="center">Shieldex</h1>

<p align="center">
  <strong>Scan extensions. Surface risk. Keep VS Code safer.</strong>
</p>

<p align="center">
  Shieldex helps you review installed VS Code extensions for suspicious behavior,
  risky permissions, dependency issues, publisher trust signals, maintenance red flags.
</p>

## Features

- Sidebar-first dashboard for fast extension security review
- One-click scan of installed extensions
- Risk score and risk level for each extension
- Detection of suspicious code patterns and dangerous behaviors
- Checks for risky capabilities, activation events, install scripts
- Publisher reputation, maintenance history, popularity, trust signals
- Dependency analysis, lockfile checks, native module detection
- OSV vulnerability scan support
- Compact extension list for quick browsing
- Detailed scan history with searchable, filterable drill-down
- Clear full history or remove one history entry
- Cancel active scan while scanning
- Export selected scan history in Markdown, JSON, HTML, CSV, SARIF
- Open extension details directly inside VS Code
